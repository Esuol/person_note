### 快捷键配置

- C -> S: Toggle Pin Tab

- C -> D: Open Command Bar

- C -> T: 快捷栏
  - Tab: 命令模式

- C -> Space: Toggle Develop Model
- C S -> Space: JS console
- C -> E: add split view
- C S -> E: close split view

- C A -> A: Open Little Arc

- C S -> S: 沉浸模式
- C S -> X: Open Extensions
- C S -> C: Copy Url

- C S -> D: Toggle toolbar
- A -> Tab: Tab 切换

1. Deepl 买 key
2. C -> G：翻译页面
3. Hover 文本 摁住 C，翻译段落
4. 输入之后 摁三下 空格，自动翻译为英文
